## git 기본 [CLI](https://ko.wikipedia.org/wiki/CLI)

[git 설치](https://git-scm.com/)

```
$ mkdir git-test
$ cd git-test
$ git init
$ echo 'hello' > README.md
$ git add README.md
$ git commit -am 'add README'
$ git push
```
