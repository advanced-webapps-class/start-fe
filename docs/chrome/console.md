# 디버깅

- Elements : HTML,CSS 코드를 분석하고, 실시간 수정 
- Console: `console.log()`, 로그 출력 
- Sources :  현재 로딩된 웹페이지에서 사용된 소스데이터 확인 
- Network : 서버와의 통신 내역을 각 리소스별로 확인 
- Application: 스토리지, 쿠키, 캐시 확인
